# South-Indian-Recipe-Book
A Flask-based South Indian Recipes web application featuring state-wise cuisines, recipe search, saved dishes, recently viewed recipes, user authentication, and an attractive responsive UI.
South Indian Recipes Web Application

A modern Flask-based web application that helps users explore authentic South Indian recipes from different states including Andhra Pradesh, Telangana, Tamil Nadu, Kerala, and Karnataka.

Features
🍽 State-wise recipe exploration
🥘 Category-based browsing (Starters, Breakfast, Lunch, Snacks, Dinner)
🔍 Smart recipe search functionality
👤 User registration and login system
❤️ Save favorite recipes
🕒 Recently viewed recipes tracking
📖 Detailed recipe pages with:
Cooking time
Ingredients
Step-by-step preparation
Recipe images
📱 Responsive and user-friendly interface
🎨 Attractive food-themed UI with animations and hover effects
Technologies Used
Python
Flask
HTML5
CSS3
JavaScript
JSON Database
